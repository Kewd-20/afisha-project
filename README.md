# Анализ пользовательских покупок


## Описание
В проекте проводится анализ поведения пользователей билетного сервиса:
изучаются повторные покупки, формируется профиль пользователей,
проверяются продуктовые гипотезы.

## Инструменты
Python, pandas, numpy, matplotlib, seaborn, SQLAlchemy.

Проект выполнен в учебной среде Яндекс Практикума
